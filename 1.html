Quick Git config — set your name & email

Click a "Copy" button to copy the command, then paste it into your Command Prompt (CMD) or terminal.

Set global username & email (applies to all repos)

git config --global user.name "Your Name"
git config --global user.email "your-email@example.com"
Use these if you want the same identity for every repo on this computer.

Set local username & email (only current repo)

cd path\to\your\project
git config user.name "Your Name"
git config user.email "your-email@example.com"
Run these inside a repository folder to override global values for that repo only.

Check your settings

git config --global --list
git config --list
Use --global --list to see global config; --list shows effective config (local overrides included).

Tip: If you use GitHub and want commits tied to your account, use the same email as your GitHub primary or add the email to your GitHub account.
